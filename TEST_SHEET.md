## F. 소유권 · 권한 (여기가 Team의 핵심 — 계정 2개로 진행)

**계정 1(tester1)로 로그인해서 파일 하나 업로드한 뒤, 로그아웃하고 계정 2(tester2)로 로그인해서 진행하세요.**

| # | 시나리오 | 절차 | 기대 결과 | 확인 |
|---|---|---|---|---|
| F1 | 남의 데이터 조회 | tester2로 로그인 후 데이터셋 목록 확인 | tester1이 올린 데이터셋도 목록에 보임(업로더 칸에 "tester1" 표시) | [ ] |
| F2 | 남의 데이터 열기 | 그 데이터셋 "열기" 클릭 | 통계·차트·집계·피벗 전부 정상적으로 볼 수 있음 | [ ] |
| F3 | 남의 데이터 태그 수정 불가 | 목록에서 tester1 데이터셋의 태그 칸 확인 | 입력칸이 아니라 읽기전용 텍스트로만 보임(수정 못 함) | [ ] |
| F4 | 남의 데이터 삭제 버튼 없음 | 목록에서 tester1 데이터셋 행의 맨 오른쪽 확인 | "삭제" 버튼 자체가 없음 | [ ] |
| F5 | 남의 데이터 열 타입 수정 불가 | tester1 데이터셋을 열어서 "열 타입" 표 확인 | 드롭다운이 아니라 읽기전용 텍스트로 표시됨 | [ ] |
| F6 | 남의 데이터에 분석저장 버튼 없음 | tester1 데이터셋에서 차트를 만들어봄 | 차트는 만들어지지만 "분석 저장" 버튼이 없음 | [ ] |
| F7 | 남의 저장된 분석에 삭제 버튼 없음 | tester1이 미리 저장해둔 분석이 있다면 "저장된 분석" 목록 확인 | "보기"는 있지만 "삭제" 버튼은 없음 | [ ] |
| F8 | 내 데이터는 정상적으로 관리 가능 | tester2 본인이 올린 데이터셋으로 위 F3~F7 다시 확인 | 전부 정상적으로 수정·삭제 가능 | [ ] |
