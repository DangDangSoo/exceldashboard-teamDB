# 사용자 퀵 가이드 — Excel2Dashboard Team

엑셀/CSV 파일을 올리면 자동으로 분석해주는 팀 공용 대시보드입니다. 코딩 지식이 없어도 됩니다.

> ⚠️ **이 서비스는 데모(MVP)입니다.** 비밀번호를 잊어버리면 되찾을 방법이 없으니 아이디·비밀번호를 꼭 기억해두세요.

---

## 1. 시작하기

### 1-1. 회원가입
1. 사이트에 접속하면 로그인/회원가입 화면이 보입니다.
2. "회원가입" 영역에 **아이디, 초대코드, 비밀번호, 비밀번호 확인**을 입력합니다.
   - 초대코드는 우리 팀 관리자에게 물어보세요(모르면 가입이 안 됩니다).
   - 아이디를 입력하면 잠시 후 옆에 "사용 가능한 아이디입니다" / "이미 사용 중인 아이디입니다"가 자동으로 뜹니다.
3. "회원가입" 버튼을 누르면 별도 로그인 없이 바로 화면이 넘어갑니다.

### 1-2. 로그인 / 로그아웃
- 이미 가입했다면 "로그인" 영역에 아이디·비밀번호만 입력하면 됩니다.
- 로그인 상태는 브라우저를 닫았다 열어도 한동안 유지됩니다.
- 화면 오른쪽 위 "로그아웃" 버튼으로 언제든 로그아웃할 수 있습니다.

---

## 2. 파일 올리기

1. 화면 아래쪽 "파일 선택" 버튼으로 엑셀(.xlsx, .xls) 또는 CSV 파일을 고릅니다.
2. 필요하면 태그를 입력합니다(콤마로 여러 개 가능, 예: `매출,2024년`). 나중에 이 태그로 모아볼 수 있습니다.
3. "업로드" 버튼을 누르면 자동으로 분석 화면이 나타납니다.

**주의**: 파일은 1행이 제목줄(헤더), 2행부터 실제 데이터라고 가정합니다. 파일 크기 10MB, 5만 행까지 지원합니다.

---

## 3. 대시보드 읽는 법

업로드하면 아래 순서로 자동 생성됩니다:

| 섹션 | 내용 |
|---|---|
| 요약 | 파일명, 행/열 수, 업로드한 사람 |
| 열 타입 | 각 열이 수치/범주/날짜/불리언 중 뭔지 자동 판별. 잘못 판별됐으면 드롭다운으로 직접 고칠 수 있음(**내가 올린 파일만**) |
| 기술통계 | 수치열은 평균·중앙값·표준편차 등, 범주열은 최빈값·상위 빈도 등 |
| 차트 | "추천 차트"는 자동으로 어울리는 걸 골라주고, "직접 만들기"로 원하는 축을 선택해 만들 수도 있음 |
| 그룹 집계 | 특정 열 기준으로 묶어서 합계/평균 등을 계산 |
| 피벗 | 행×열 교차표 |
| 저장된 분석 | 예전에 저장해둔 차트/집계/피벗을 다시 볼 수 있는 곳 |
| 미리보기 | 원본 데이터 상위 20행 |

---

## 4. 차트 저장하고 다시 보기

마음에 드는 차트/집계/피벗을 나중에도 다시 보고 싶다면:

1. 결과 아래 "분석 저장" 버튼을 클릭 (내가 올린 데이터셋에서만 보임)
2. "저장된 분석" 목록에 새 항목으로 추가됨
3. 나중에 "보기"를 누르면 **최신 데이터 기준으로 다시 계산해서** 보여줍니다(그림 파일을 저장해두는 게 아니라, 매번 새로 그림).

각 차트는 "PNG 다운로드"로 이미지 파일을 내려받을 수 있습니다.

---

## 5. 태그로 모아보기

- 데이터셋 목록 위 "태그로 모아보기" 칸에 태그를 입력하고 "필터"를 누르면 그 태그가 붙은 파일만 보입니다.
- "전체보기"를 누르면 다시 전체 목록이 보입니다.
- 내가 올린 데이터셋의 태그는 목록에서 직접 수정하고 "태그 저장"을 누르면 됩니다.

---

## 6. 팀원들과 데이터 공유하기

- **로그인한 팀원이라면 누구나 서로의 데이터셋을 볼 수 있습니다** — 목록·통계·차트·집계·피벗 전부 조회 가능.
- 단, **수정·삭제는 그 파일을 올린 사람만** 할 수 있습니다. 다른 사람이 올린 데이터셋은:
  - 태그 칸이 수정 안 되는 읽기 전용 텍스트로 보임
  - "삭제" 버튼이 아예 없음
  - 열 타입도 드롭다운이 아니라 텍스트로만 보임
  - "분석 저장" 버튼이 없음(조회는 되지만 내 걸로 저장은 못 함)

즉, "다 같이 보고, 올린 사람만 관리한다"는 원칙입니다.

---

## 7. 내 데이터 정리하기

- 데이터셋 목록에서 내가 올린 항목의 "삭제"를 누르면 확인창이 뜨고, 확인하면 완전히 삭제됩니다(되돌릴 수 없음).
- 저장해둔 분석도 같은 방식으로 개별 삭제할 수 있습니다.
- 데이터셋을 삭제하면 그 안에 저장해둔 분석도 함께 정리됩니다.

---

## 8. 자주 묻는 질문

**Q. 비밀번호를 잊어버렸어요.**
A. 이 서비스는 데모라 자체 복구 기능이 없습니다. 관리자에게 문의하세요(서버 데이터베이스를 직접 확인해야 합니다).

**Q. 업로드가 안 돼요.**
A. 파일 형식(xlsx/xls/csv)과 크기(10MB 이하), 행 수(5만 행 이하)를 확인해주세요. 1행은 반드시 제목줄이어야 합니다.

**Q. 다른 사람이 올린 파일을 지우고 싶어요.**
A. 안 됩니다(의도된 동작입니다). 필요하면 그 사람에게 직접 요청하세요.

**Q. 회원가입이 안 돼요.**
A. 초대코드가 틀렸거나, 이미 있는 아이디일 수 있습니다. 화면에 뜨는 에러 메시지를 확인해주세요.
